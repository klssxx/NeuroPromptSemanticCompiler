# Guía rápida

1. Ejecuta `./run_gui.sh`.
2. Abre la sección **Compilador**.
3. Pega tu prompt original.
4. Deja `AUTO` como tipo de mejora.
5. Pulsa **Mejorar prompt**.
6. Abre **Resultados** y copia **Prompt listo para usar**.
7. Revisa **Validación** para comprobar la puntuación de preservación.

Para usar el resultado en otra IA, copia **Prompt listo para usar**.
Los archivos técnicos son opcionales. No necesitas comprenderlos para mejorar un prompt.
Si no entiendes una palabra, abre **Glosario** en el menú lateral.

La aplicación funciona offline y no envía prompts a servidores externos.
