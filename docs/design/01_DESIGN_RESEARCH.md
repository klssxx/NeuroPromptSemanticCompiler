# 01 — Design Research

## Key References

### KDE Breeze Design System
- Flat surfaces with subtle 1px borders
- Blue accent (#3DAEE9) — we use a tempered cyan (#31B8D6) as our identity accent
- Clear light/dark mode separation
- Minimal gradients; solid fills preferred
- Focus indicators: 2px blue outline
- Card elevation via background tint, not shadows

### Kate / KDevelop / Dolphin
- Sidebar navigation with icon+text
- Flat header bars
- Status bar with muted text
- Tab bars with clear active state

### VS Code / JetBrains (principles, not copy)
- Activity sidebar collapses to icons
- Settings gear in header
- Clear empty state illustrations

## Transferable Principles
1. Solid backgrounds > gradients for performance
2. 1px borders with muted colors for grouping
3. Clear active/selected states on nav
4. Focus rings: 2px solid accent outline
5. Empty states with placeholder text + icon
6. Light mode: white/near-white bg, dark text; Dark mode: dark bg, light text
7. Accent color used sparingly (primary actions only)
8. Surface hierarchy: base → card → elevated (each slightly lighter/darker)

## What NOT to Copy
- KDE's specific blue (#3DAEE9) — we keep our cyan identity but tone it
- KDE's specific font sizes — we keep our current sizing
- GNOME Adwaita roundedness or libadwaita patterns
