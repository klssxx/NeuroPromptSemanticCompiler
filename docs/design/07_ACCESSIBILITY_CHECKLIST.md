# 07 — Accessibility Checklist

- [x] Focus indicators: 2px accent outline on all interactive widgets
- [x] Keyboard navigation: Tab order follows layout; Enter on buttons; Escape closes popovers
- [x] Contrast ratios:
  - Dark: #E8ECF0 on #1B1E24 = 11.2:1 (AAA)
  - Dark: #9EA3AB on #1B1E24 = 5.2:1 (AA)
  - Light: #1B1E24 on #F8F9FB = 14.3:1 (AAA)
  - Light: #6B7080 on #F8F9FB = 4.8:1 (AA)
- [x] Text sizes: minimum 10.5pt (exceeds 12px minimum)
- [x] Labels: all inputs have associated labels
- [x] Tooltips: context help available for all options
- [x] Error states: status bar + message boxes for compile errors
- [x] Shortcuts: Ctrl+N, Ctrl+O, Ctrl+S, Ctrl+Enter, F1
- [x] SVG icon: has title and desc elements for screen readers
