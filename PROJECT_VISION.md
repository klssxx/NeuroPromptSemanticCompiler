# NeuroPrompt Semantic Compiler Vision

No transmitir palabras, transmitir intención estructurada.

NeuroPrompt Semantic Compiler trata el prompt como programación de contexto artificial: convierte lenguaje humano ruidoso en estructura operativa verificable.

## Principios

- Prompt as artificial context programming.
- Semantic seeds as reusable prompt DNA.
- Más poder por token sin perder intención crítica.
- Verificación explícita de pérdida de contexto antes de aceptar compresión.
- Safety-first: restricciones críticas nunca se eliminan.

## Alcance MVP

- Diccionario semántico + lenguaje NSL + reconstrucción + verificación.
- Sin píxeles/audio/señales físicas en esta versión.
- Sin APIs externas ni ejecución remota.
